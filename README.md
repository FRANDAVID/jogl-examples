# Computer Graphics with JOGL

Computer Graphics with JOGL (Java Binding for the OpenGL)

### Examples
1. Line
2. Triangle
3. Circle
4. Cuble
5. Rhombus
6. etc

### How to Play
```
mvn compile exec:java -Dexec.mainClass="cg.Cube"
```
